## 目录职责

| 目录 | 职责 |
| --- | --- |
| `project/` | 项目状态、SOP、Sprint、Handoff、文档规范、P0 验收 |
| `product/` | PRD、需求池、评分系统 |
| `architecture/` | 架构、API contract、数据流 |
| `agents/` | 多 Agent 写作与开发协作 |
| `templates/` | 任务、Agent、Review 模板 |

## 文档原则

- README 面向首次使用者。
- `操作文档.md` 面向实际操作者。
- `docs/product/PRD.md` 是产品范围真相源。
- `docs/project/HANDOFF.md` 是当前交接真相源。
- `devlog/` 是历史事实证据，不随意删除。
